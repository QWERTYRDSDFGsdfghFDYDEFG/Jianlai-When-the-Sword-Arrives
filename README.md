# 📖 项目简介

**When the Sword Arrives** 是一个以《剑来》为原作背景的视觉小说项目，  
主要用于：

- 剧情演出练习
- 视觉小说开发流程探索
- Galgame 叙事结构研究
- 分支剧情与角色塑造实践

本作以《剑来》“书简湖篇”为主线改编，  
时间线从：

> **陈平安离开大隋书院分别之后**  
> 到  
> **崔巉布下书简湖问心局**

相比传统强调“升级”“碾压”的叙事，本项目更关注：

- 人在规则中的选择
- 情理与规矩之间的冲突
- 亲疏远近带来的判断偏差
- 普通人在秩序保护下的生存空间
- 成长不可避免的代价

希望通过剧情、分支与演出，让这些问题真正落到玩家手里。

# ✨ 项目特色

## 🎭 剧情驱动演出

- 多角色视角叙事
- 对话分支系统
- 情绪化镜头演出
- 氛围推进型节奏设计
- 多阶段情绪反馈

---

## 🧠 更偏“思考”的剧情结构

项目不会单纯强调战力强弱，  
而是更关注：

- 规则与人情
- 对错与立场
- 善意的代价
- 成长中的失去
- “问心”本身

---

## 🎧 原作氛围还原

项目部分配音与 BGM 参考《剑来》广播剧，  
尽可能保留书简湖篇压抑、沉静、复杂的氛围。

---

## 🌊 沉浸式文本体验

项目将重点强化：

- 台词节奏
- 留白感
- 情绪停顿
- 环境氛围
- 人物关系张力

尽量还原《剑来》原作中“慢”和“沉”的气质。
