# script_chapter2 ~ script_chapter5 逐场分镜表

> 用途：将 `script_chapter2.rpy` 到 `script_chapter5.rpy` 拆成可制作的 VN 分场演出表。  
> 目标：每一场都能直接落到“背景、立绘、表情、站位、景别、演出备注”。  
> 口径：优先复用现有立绘与背景，不追求镜头数量堆积，追求低成本、高识别度、叙事清晰。

---

## 使用规则

- `背景` 以现有或待补的场景图为单位，不要求每场一张新图。
- `立绘` 默认优先使用现有陈平安标准体系；其他角色可先用占位版本。
- `表情` 先用大类：平静、温和、疲惫、警觉、讲理、压抑、看穿不说破。
- `站位` 以 `左 / 中 / 右` 为主，必要时用 `偏左 / 偏右 / 前景压屏`。
- `景别` 以 `远景 / 中景 / 近景 / 特写` 为主，VN 中主要靠立绘缩放完成。
- `演出备注` 重点说明是否压暗未发言角色、是否切轻环境、是否要转场或沉默留白。
- 每场先判断 `图像类型`：`基础立绘场 / 动作立绘场 / 事件构图图`。
- 每场再判断 `动作级别`：`A0 标准静立 / A1 轻动作对白 / A2 关系动作 / A3 事件动作`。
- 凡是 `A2 / A3` 场次，都必须写清 `动作设计` 和 `禁区`，防止生成图退化成普通立绘。
- 通用动作判断、镜头摆位与素材复用规则，统一参考 [《剑来》VN演出与构图总手册.md](/E:/新建文件夹/When%20the%20Sword%20Arrives/game/《剑来》VN演出与构图总手册.md:1)。

### Chapters 2-5 统一执行口径

- `Chapter 2` 重点是归路、入城、见闻，动作以 `A1 / A2` 为主，少量奇观 `A3`
- `Chapter 3` 重点是宴席说理和夜谈，动作以 `A1 / A2` 为主，不要无谓大动作
- `Chapter 4` 重点是平行线切入和顾璨将出场，远望、试探、观局比热闹更重要
- `Chapter 5` 重点是陈平安与顾璨的关系压迫，必须把“谁在逼问、谁在承受、谁在退让”做出来

## Chapter 2

### 场次 2-01 返乡山路与溪涧歇脚

| 项目 | 内容 |
|---|---|
| 剧情节点 | 陈平安、裴钱、朱敛、石柔在返乡路上歇脚，挑水疱，谈回乡和将来 |
| 背景 | 山野小径 + 溪涧石岸 |
| 图像类型 | 动作立绘场 |
| 动作级别 | `A2 关系动作` |
| 立绘 | 陈平安标准版、裴钱日常版、朱敛日常版、石柔静立版 |
| 表情 | 陈平安平静/带心事；裴钱活泼；朱敛促狭；石柔冷静 |
| 站位 | 陈平安中偏左，裴钱右前，朱敛左后，石柔右后 |
| 动作设计 | 陈平安更像歇脚后站稳、手上有处理水疱或扶膝后的收势；裴钱不该纯站定，带一点蹲起、探身或凑近说话的孩子动作；朱敛、石柔作为后景托住队伍关系。 |
| 禁区 | 不能画成四个人平均站位看镜头；归路感必须来自“停下歇脚”，不是站场。 |
| 景别 | 中景为主，开场先远景后切中景 |
| 演出备注 | 套 `开场 + 日常对话` 套板。先用远景铺“归路清静”，再切中景做人物关系。适合 very light 薄雾。 |

## 制作优先级建议

### 第一批必须先做

- 陈平安标准立绘与基础表情
- 陈平安书简湖版表情
- 吴懿基础立绘
- 顾璨基础立绘
- 返乡山路背景
- 黄庭郡城背景
- 紫阳府外广场 / 宴厅 / 多宝楼背景
- 池水城古董铺背景
- 书简湖湖面 / 青峡岛道路背景

### 第二批建议补

- 孙大侠基础立绘
- 小泥鳅基础立绘
- 刺客妇人立绘
- 阮秀 / 宋郎中 / 杨老头占位立绘
- 楼船奇观图
- 战乱官道背景

### 最低成本落地原则

- 同一章内尽量复用 2 到 4 张背景。
- 同一场内尽量用 2 到 3 个景别，不要无限切镜。
- “讲理戏”优先靠表情差分和轻微缩放，不靠复杂动作。
- “压抑戏”优先靠留白、停顿、未发言角色压暗，不靠过度特效。
- 顾璨线的核心不是大场面，而是“陈平安说话时，顾璨怎么听”。
