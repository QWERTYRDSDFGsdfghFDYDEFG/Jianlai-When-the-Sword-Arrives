# When the Sword Arrives

> 基于《剑来》书简湖篇改编的 2D 叙事游戏 / 视觉小说原型。  
> 核心体验不是战力成长，而是陪陈平安走入一场关于规矩、人情、选择与代价的问心局。

---

## 项目状态

当前项目处于 **Ren'Py 可玩原型开发阶段**。

已经具备：

- 基础 Ren'Py 工程结构
- 主菜单、设置、存档、读档等默认流程
- 从 `label start` 到 `chapter12_start` 的完整主线跳转链
- 多章主线剧情脚本
- 部分背景、角色图与音频素材
- 情绪 / 状态相关逻辑雏形

仍在建设：

- 章节节奏与断点重排
- 分支选择与状态反馈
- 角色立绘、背景、CG 与音频补齐
- 第一版可玩 Demo 的完整验收

## 故事范围

当前主线以《剑来》书简湖篇为核心，时间线大致从：

> 陈平安离开大隋山崖书院  
> 到  
> 书简湖问心局结束，陈平安带顾璨走出一段路后继续北归

重点内容包括：

- 山崖书院分别后的归乡路
- 入书简湖前的顾璨危机预警
- 顾璨与小泥鳅在书简湖的恶名
- 青峡岛饭桌、春庭府母子、顾璨问心
- 刘志茂、刘老成、刘重润、马远致、红酥等书简湖人物线
- 曾掖、马笃宜与阴物夙愿线
- 大骊、朱荧王朝、外乡宗门对书简湖大势的挤压
- 崔瀺、崔东山、阮秀与齐静春赌局的上层揭示

## 剧情整理结论

现有脚本跳转完整：

```text
script.rpy
  -> script_chapter2.rpy
  -> script_chapter3.rpy
  -> script_chapter4.rpy
  -> script_chapter5.rpy
  -> script_chapter6.rpy
  -> script_chapter7.rpy
  -> script_chapter8.rpy
  -> script_chapter9.rpy
  -> script_chapter10.rpy
  -> script_chapter11.rpy
  -> script_chapter12.rpy
```

但从剧情节奏看，现有“文件章节”不完全等于最终“游戏章节”。建议将前期较短的铺垫章节合并为序章与入湖前篇，将后期过长的第 10、11、12 章拆成多个可游玩章节。

### 可以合并的章节

| 现有文件 | 建议 | 原因 |
| --- | --- | --- |
| `script.rpy` | 保留为序章 | 山崖书院送别完整，情绪功能独立，是开场钩子。 |
| `script_chapter2.rpy` + `script_chapter3.rpy` | 合并为“归乡路上” | 都是离开书院后的返乡与入湖前铺垫，核心是陈平安、裴钱、朱敛的路上关系与黄庭国见闻。 |
| `script_chapter4.rpy` 前半 | 接入“归乡路上”结尾 | 红烛镇、顾韬预警、陈平安决定改道书简湖，是入湖前篇的自然收束。 |

### 需要拆出来的章节

| 现有文件 | 建议 | 原因 |
| --- | --- | --- |
| `script_chapter4.rpy` 后半 | 拆为“书简湖初见” | 石毫国战乱、阮秀线、顾璨刺杀现场与两巴掌，是正式入湖的强事件，应独立成章。 |
| `script_chapter7.rpy` | 可拆为“锦囊”与“红酥”两个小章 | 前半是崔瀺 / 崔东山高层问局，后半是红酥人物线，情绪质地不同。 |
| `script_chapter10.rpy` | 拆为“亡魂账本”“小泥鳅问心”“刘志茂交易”“雪中顾璨” | 单文件容量很大，且有多个独立高潮。 |
| `script_chapter11.rpy` | 拆为“宫柳岛问刘老成”“石毫国阴物年关” | 刘老成谈红酥与周过年阴物线主题不同，分开更清楚。 |
| `script_chapter12.rpy` | 必须拆分 | 现有第 12 章包含披云山山盟、鹘落山、书简湖大变、三返书简湖、顾璨新约、母子离湖、崔瀺揭局、顾璨同行、齐静春尾声，容量远超单章。 |

以下是建议建立的最终章节结构。括号中为对应现有脚本来源。

| 最终章 | 章节名 | 来源 | 本章主题 |
| --- | --- | --- | --- |
| 序章 | 山崖别离 | `script.rpy` | 以李宝瓶、李槐、裴钱等人的送别演出开场，建立“离别之后仍有牵挂”的情绪底色，陈平安第一次承认崔东山是学生。 |
| 第一章 | 归乡路上 | `script_chapter2.rpy` + `script_chapter3.rpy` + `script_chapter4.rpy` 前半 | 陈平安带裴钱、朱敛、石柔返乡，黄庭国、紫阳府、顾韬等见闻逐步提示世道与规则的复杂，并引出“必须先去书简湖”。 |

| 第三章 | 青峡岛饭桌 | `script_chapter5.rpy` | 陈平安见顾璨母子，借刺客、饭桌与泥瓶巷旧事试探顾璨的善恶边界；主题是“活着的苦，能不能成为杀人的理由”。 |
| 第四章 | 心中杀刀 | `script_chapter6.rpy` | 崔瀺与崔东山旁观问局，陈平安与顾璨夜谈，顾璨承认自己喜欢杀人；主题是“讲理失败之后，道理反过来刺向自己”。 |
| 第五章 | 请你不要失望 | `script_chapter7.rpy` 前半 | 裴钱锦囊成为陈平安最低处的支撑，崔瀺揭示问心局难处；主题是“对世界失望之前，先看见仍有人把你拉住”。 |

## 当前脚本文件建议

短期不必立刻改动脚本文件名和跳转，可以先在 README 中用最终章节规划指导后续整理。等剧情稳定后，再考虑把脚本拆成更接近最终章节的文件：

```text
script_prologue.rpy
script_chapter01_return_road.rpy
script_chapter02_shujian_lake_first_meet.rpy
...
script_chapter20_slow_return.rpy
```

拆分时优先处理：

1. `script_chapter12.rpy`，体量最大，剧情阶段最多。
2. `script_chapter10.rpy`，存在多个独立高潮。
3. `script_chapter11.rpy`，刘老成线与阴物年关线应分章。
4. `script_chapter7.rpy`，高层问局与红酥小传可分开。

合并时优先处理：

1. `script_chapter2.rpy` 与 `script_chapter3.rpy`。
2. `script_chapter4.rpy` 前半接入入湖前铺垫。

## 本地运行

1. 安装 Ren'Py。
2. 用 Ren'Py Launcher 导入本项目目录。
3. 启动项目，选择 `Launch Project`。

建议测试时重点检查：

- 新游戏能否从 `label start` 正常进入
- 章节跳转是否断链
- 角色名是否未定义
- 是否有缺图、缺音频或乱码
- 存档、读档、快进、回看、设置菜单是否正常

---

## 开发约束

- 不随意删除已有剧情与素材。
- 修改剧情时优先保持当前章节连续性。
- 新增角色先在 `characters.rpy` 定义。
- 长叙述拆成适合 VN 阅读的短旁白。
- 每次大改章节结构后，检查 `jump` 链和 Ren'Py 语法。
- 未来拆分脚本文件时，应同步更新本 README 的章节规划。
