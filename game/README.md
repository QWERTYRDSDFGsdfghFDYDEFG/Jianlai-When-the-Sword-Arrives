> 一款基于《剑来》书简湖篇改编的 2D 叙事游戏 / 视觉小说。  
> 在规矩、人情、选择与代价之间，陪陈平安走进一场缓慢而沉重的问心局。

## 给玩家

**When the Sword Arrives** 不是一款强调数值成长和战力碾压的游戏。

它更想呈现的是：

- 一个外乡人进入书简湖后，如何理解这里的规矩
- 一个少年面对旧友、亲情、善恶与因果时，如何做选择
- 一个看似混乱的江湖，为什么会用自己的方式运转
- 当善意不足以解决问题时，人还能不能守住本心

你会读到大量克制的对白、沉默的停顿、细碎的人情往来，以及一些并不轻松的判断。  
如果你喜欢慢热、重文本、重人物关系的叙事体验，这个项目就是朝着那个方向做的。

## 故事范围

当前主线以《剑来》书简湖篇为核心，时间线大致从：

> 陈平安离开大隋书院之后  
> 到  
> 崔巉布下书简湖问心局

重点内容包括：

- 大隋分别后的行路
- 初入书简湖
- 顾璨与青峡岛
- 刘志茂、刘老成、刘重润、马远致等人物线
- 陈平安对书简湖规矩的观察与问心
- 书简湖大局推进

---

## 核心体验

本项目希望做出一种偏沉浸式阅读的视觉小说体验：

- **慢节奏文本**：让情绪和关系自然沉下来。
- **角色立场冲突**：不急着替玩家下结论。
- **选择与代价**：后续会逐步加入影响关系、状态和剧情反馈的选择。
- **书简湖氛围**：压抑、复杂、讲规矩，却又处处不讲理。
- **问心主题**：真正的核心不是赢，而是看清自己为何要这样做。

## 本地运行

1. 安装 Ren'Py。
2. 用 Ren'Py Launcher 导入本项目目录。
3. 启动项目，选择 `Launch Project`。

建议测试时重点检查：

- 新游戏能否从 `label start` 正常进入
- 章节跳转是否断链
- 角色名是否未定义
- 是否有缺图、缺音频或乱码
- 存档、读档、快进、设置菜单是否正常
