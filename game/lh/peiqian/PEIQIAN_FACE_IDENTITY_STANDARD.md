# 裴钱固定身份与当前服装标准

适用对象：`lh/peiqian/` 目录下裴钱的全部后续立绘、表情、动作版、章节版、场景版。

母版基准图：

- 当前身份与服装母版：`peiqian_master_v02.png`

## 1. 目标定义

本标准的目标，不是每次重新画一个“可爱的女孩”，而是固定裴钱当前可复用的母版身份。

后续一切变化，必须建立在同一个裴钱的前提上进行：

- 年龄感固定
- 脸部路子固定
- 体型固定
- 发型固定
- 气质底色固定
- 当前服装固定

允许变化的内容只有：

- 表情
- 动作
- 场景

禁止把每一张新图都当成“重新设计一次裴钱”。

## 2. 六项硬锁定

以下六项是后续所有生成的最高优先级，任何一项漂移，都视为不合格。

### 2.1 年龄感锁定

- 裴钱必须明显是小姑娘、小徒弟，不是成熟少女。
- 年龄感应稳定在“尚未长开”的阶段，第一眼就要比成年女性更小。
- 不允许画成成熟女主、青年女主、古偶女主脸。

### 2.2 脸部路子锁定

- 以 `peiqian_master_v02.png` 为当前唯一脸路子母版。
- 脸型必须是小脸、短下庭、偏圆润的少女骨架，整体更接近“未长开的小徒弟”，不是尖下巴网红脸，也不是成熟鹅蛋脸。
- 眉眼是当前母版最核心的识别点：眉毛偏自然平直，眼型是略大的杏眼，但眼神必须清亮、专注、带一点机灵和不服输，不能画成过圆的萌眼、无辜甜妹眼或媚态女主眼。
- 鼻子要小而收，鼻梁存在感轻，不做精修立体高鼻；重点是干净、自然、耐看，而不是精致攻击性的五官雕刻感。
- 嘴部必须小、薄厚适中、常态偏闭合或轻微克制上扬，像心里有主意的小姑娘，不能画成饱满唇、艳唇、咧嘴甜笑或偶像式营业表情。
- 整张脸的完成效果应当是“灵、稳、野气里带分寸”，第一眼让人觉得她聪明、能吃苦、爱逞小强，不是被包装过的漂亮小女主。

原则：可以换表情，但不能换成“甜妹女主”脸。

### 2.3 体型锁定

- 体型必须偏小、偏瘦、结实。
- 肩和上身要收，不要画成成熟比例，也不要画得过于肉感。
- 要有“能走长路、能吃苦、能蹲下干活”的小江湖徒弟感。

### 2.4 发型锁定

- 发型固定为实用型双髻 / 双团发。
- 不允许改成披发、华丽发髻、贵族小姐头、女主精致编发。
- 发饰必须克制、简朴，不可出现贵气头饰和复杂金饰。

### 2.5 气质底色锁定

裴钱的底色必须始终是：

- 机灵
- 利索
- 能吃苦
- 专注
- 小徒弟感

允许在具体画面里出现：

- 雀跃
- 认真
- 小警觉
- 低头做事时的专注
- 跟在师父身边时的松快

但不允许整体气质改成：

- 成熟女主感
- 温柔古偶女主感
- 大家闺秀感
- 妩媚感
- 甜妹偶像感

原则：她可以活泼，但不是“女主角式的活泼”。

### 2.6 当前服装锁定

- 以 `peiqian_master_v02.png` 为当前唯一服装母版。
- 当前阶段默认固定为深色朴素短打 / 行路装。
- 服装必须保持以下结构关系：
  - 深黑蓝或暗靛色主色
  - 朴素短打
  - 便于行动的层次结构
  - 收紧袖口与腰部
  - 绑腿
  - 朴素布鞋
- 衣服可以有轻微旧感、风尘感、耐磨感，但不允许换成精致女装、飘逸裙装、贵族衣装。

原则：可以旧一点，但不能换装成“女主衣柜”。

## 3. 允许变化与禁止变化

### 3.1 允许变化

- 表情变化：平静聆听、雀跃轻快、小警觉、低头认真、压住委屈、憋着小得意
- 动作变化：站立、并步跟随、侧身听话、低头做事、蹲下、轻微探身、回头
- 场景变化：山路、溪边、渡口、郡城街口、舟上、屋内、夜路、雨雾

### 3.2 禁止变化

- 禁止画成熟
- 禁止画成女主脸
- 禁止画成温柔甜妹
- 禁止画成大家闺秀
- 禁止过圆大眼、过萌脸、过甜笑
- 禁止肩膀变宽、身段变成熟
- 禁止脱离 `peiqian_master_v02.png` 另起一套服装
- 禁止改成浅色飘逸衣裙
- 禁止改成贵气、整洁、讲究的女主穿搭
- 禁止华丽发饰、复杂首饰、精修偶像风

## 4. 生成提示词总规则

后续任何裴钱出图，都必须默认附带以下约束：

```text
以 peiqian_master_v02 为唯一当前身份与服装母版。
严格锁定裴钱的年龄感、脸部路子、体型、双髻发型、气质底色和当前服装版本。
必须明显是小姑娘、小徒弟，而不是成熟少女或女主脸。
必须保持偏小、偏瘦、结实、利索、能吃苦的体型与气质。
服装必须保持 peiqian_master_v02 的深色朴素短打和行路装结构，不允许擅自换装。
只允许修改表情、动作和场景。
如果新图更好看但不像 peiqian_master_v02，或服装偏离 peiqian_master_v02，也判定为失败。
```

## 5. 生成时的负面约束

后续生成时，应同时加入以下否定约束：

```text
不要成熟少女，不要成熟女主，不要温柔女主脸，不要甜妹偶像感，不要大家闺秀，不要华丽发饰，不要贵气衣装，不要浅色飘逸裙装，不要圆大萌眼，不要成人比例，不要浪漫女主氛围，不要精修古偶风。
```

## 6. 出图验收清单

每次新图生成后，都必须按以下顺序验收：

1. 第一眼是否还是 `peiqian_master_v02` 那个裴钱，而不是另一个古风小女孩。
2. 年龄感是否仍然是小姑娘、小徒弟，而不是成熟少女。
3. 脸部和眼神是否仍然偏收、偏专注，而不是圆眼甜妹。
4. 体型是否仍然偏小、偏瘦、结实。
5. 发型是否仍然是实用型双髻。
6. 服装是否仍然是 `peiqian_master_v02` 的深色朴素短打版本。
7. 气质是否仍然是机灵、利索、能吃苦，而不是女主化。

只要第 1-4 项里有任一项不稳定，或第 6 项服装跑偏，就直接判定废弃，不进入后续使用。

## 7. 文件与版本约定

- `peiqian_master_v02.png` 是裴钱当前固定身份与服装母版。
- 后续所有裴钱图，默认都应注明“基于该母版锁身份并锁服装生成”。
- 新图命名应表达“变化项”，而不是暗示“重新做人设”。

推荐命名思路：

- `peiqian_master_v02_exp_xx.png`
- `peiqian_master_v02_act_xx.png`
- `peiqian_master_v02_scene_xx.png`

## 8. 执行原则

结论只有一条：

裴钱后续所有图像工作，首先是在维护同一个小徒弟裴钱的身份连续性与当前服装连续性，其次才是做单张画面的表现力。
