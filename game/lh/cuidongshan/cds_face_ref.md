# 崔东山固定脸部身份与当前服装标准

适用对象：`lh/cuidongshan/` 目录下崔东山的全部后续立绘、表情、动作版、章节版、场景版。

当前唯一主标准文件：`lh/cuidongshan/cds_face_ref.md`

当前主母版图：

 - `lh/cuidongshan/cds.png`
 - `lh/cuidongshan/cds_16view_v1.png`（角度辅助母版）
 - `lh/cuidongshan/cds_master_walk_v1_prompt.md`（单图母版提示词，按需单独使用）

母版基准图：

 - `lh/cuidongshan/cds.png`
 - `lh/cuidongshan/cds_16view_v1.png`


执行原则：

- 以后查崔东山标准，只看这一份。
- 后续一切出图，先锁同一个人，再谈场景、情绪和构图。
- 这份文件同时覆盖原先的锁脸说明、服装口径、提示词约束、高压场景提醒和情绪版主母版口径。
- 新测试图或候选图默认不触发本文件更新；只有正式替换当前标准母版时才同步修改。

## 1. 目标定义

本标准的目标，不是每次重新画一个“更好看的白衣少年”，而是固定崔东山当前可长期复用的同一人物身份，以及当前阶段统一服装版本。

后续所有变化，都必须建立在以下前提上：

- 脸型固定
- 五官比例固定
- 年龄感固定
- 发型轮廓固定
- 身形体态固定
- 气质底色固定
- 当前服装固定

允许变化的内容只有：

- 表情
- 动作
- 场景
- 景别
- 轻微光线关系

禁止把每一张新图都当成“重新设计一次崔东山”。

## 2. 七项硬锁定

以下七项是后续所有生成的最高优先级，任何一项漂移，都视为不合格。

### 2.1 脸型锁定

以 `lh/cuidongshan/cds.png` 为唯一主脸型母版，`lh/cuidongshan/cds_16view_v1.png` 为角度辅助母版。
- 脸部整体必须偏窄、偏长、收束明确，不能变圆，不能变宽，也不能削成夸张 V 脸。
- 额头、颧侧、下颌、下巴的收放关系必须保持母版一致。
- 下颌不能成熟化，不能贵公子化，不能偶像脸化。

### 2.2 五官比例与识别点锁定

- 眉眼距离、眼裂长度、鼻口落点、耳位、发际线关系固定。
- 眼睛必须偏收、略长，不圆、不甜、不空。
- 鼻梁细直克制，嘴唇偏薄，嘴角自然收束，不能改成温柔男主嘴型。
- 眉心红籽必须始终保留，而且要小、凝、清楚可辨；不能丢、不能糊、不能画成夸张花钿。

原则：可以换表情，但不能换五官设计。

### 2.3 年龄感锁定

- 固定年龄感为 `18-21` 岁的青年男子。
- 必须保留白衣少年感和书卷气，但不能幼化成稚气少年。
- 不能成熟化成世家公子、标准男主或城府过重的成年谋士脸。
- 不要过度精修，不要偶像化，不要美妆感。

### 2.4 发型轮廓锁定

- 发型固定为黑色长发半束 / 半挽高髻。
- 额前与两侧要保留轻微散乱感，不能梳成整齐贵气发式。
- 不允许改成披发公子头、复杂高冠、华丽头饰或贵族礼冠。

### 2.5 身形与体态锁定

- 身形必须修长、偏轻、不壮、不厚，不是武夫体格。
- 体态要有读书人气，外壳可以略松、略懒，但骨子里不能塌。
- 不允许画成威武英雄姿态，也不允许画成病弱美人或阴柔妖异路线。

### 2.6 气质底色锁定

崔东山的底色必须始终是：

- 外轻内重
- 清醒克制
- 孤独深情
- 古怪但有分寸
- 极聪明，但不是炫耀聪明

允许在具体画面里出现：

- 似笑非笑
- 旁观感
- 轻微讥诮
- 安静注视
- 对陈平安时少见的认真、松快、维护感

但不允许整体漂成：

- 温柔男主
- 古偶美男子
- 贵公子
- 仙侠偶像
- 邪魅反派
- 纯搞笑角色

### 2.7 当前服装锁定

- 以 `lh/cuidongshan/cds.png` 为当前唯一通用服装母版。
- 当前阶段默认固定为“白色外袍 + 灰蓝内衫”的白衣读书人装束。
- 必须保持以下结构关系：
  - 白色长外袍
  - 宽大衣袖与自然下垂的衣摆
  - 灰蓝色内层长衫
  - 简洁克制的领口与前襟层次
  - 低调的前身垂饰 / 穗带关系
  - 黑色布鞋
- 常规中性状态图可默认保留卷轴口径；若剧情需要空手或换道具，也只能改道具，不能顺势改掉整套服装逻辑。
- 衣物可以有轻微褶皱、风感和使用痕迹，但不能换成金玉堆砌的贵气装、仙侠飘带装或戏剧化礼服。

原则：可以有场景变化，但不能脱离 `lh/cuidongshan/cds.png` 另起一套通用服装。

## 3. 允许变化与禁止变化

### 3.1 允许变化

- 表情变化：平静、淡笑、似笑非笑、略带讥诮、认真、安静看人、若有所思
- 动作变化：站立、侧身、回望、执卷、垂眸、抬眼、偏头、缓步、倚坐
- 场景变化：书院、湖岸、舟上、廊下、屋内、山路、夜色、案前、官道
- 光线变化：暖灯、月色、阴天、逆光边缘光，但只能改变氛围，不能借机改脸

### 3.1.1 表情差分必须写成结构变化

后续不要只写“淡笑”“认真”“若有所思”。崔东山这类角色如果只给抽象情绪词，最容易被模型画成几乎一样的温和男主脸。

每次表情变化，至少同时明确：

- 眉的状态
- 眼睑开合与视线方向
- 嘴角和唇线状态
- 下颌和面部收紧程度
- 头部细微偏转角度

推荐写法示例：

- `似笑非笑：眉眼放松但带观察感，眼尾轻收，视线像在看人又不完全交底，嘴角单侧极轻上提，唇线仍薄，下颌放松，头微偏。`
- `认真：眉头轻收但不压狠，上眼睑略压，目光集中不飘，嘴角收平，下颌轻收，头略前倾。`
- `略带讥诮：一侧眉略提，眼神带一点冷静判断，嘴角轻微偏斜，不要咧嘴，下颌稳定，头轻偏。`
- `若有所思：眉眼整体收静，视线微微下落或停在远处，嘴角闭合，面部张力收住，头小幅低垂。`

### 3.1.2 动作去僵硬写法

后续不要只写“侧身”“回望”“缓步”。这会直接导向最普通的立绘待机姿势。

每次动作变化，至少同时明确：

- 重心落在哪条腿
- 另一条腿是在前探、后撤还是平衡
- 肩线是否倾斜
- 左手和右手分别在做什么
- 头转向哪边、转了多少
- 衣摆、发尾、袖口是否有轻微跟随

推荐写法示例：

- `半侧身站立，重心落在左脚，右脚向后撤半步，肩线轻微右低左高；左手执卷贴近身侧，右手自然放松；头向右回望约二十五度；外袍下摆和发尾有轻微跟随。`
- `缓步前行但不是大跨步，重心仍主要落在后脚，前脚只轻轻探出；躯干保持修长稳定，双臂顺着步态做很轻的摆动；头略偏向前方侧面，衣摆有轻微流动，不要夸张摆拍。`
- `案前停步，身体正中略偏半侧，重心平均但肩线不完全水平；一手按卷，一手垂落；头微低，视线抬起看人，动作克制但不是僵直站桩。`

### 3.2 禁止变化

- 禁止重新设计脸型
- 禁止把眼睛放大来换“精致感”或“少年感”
- 禁止把嘴唇画厚、画软、画成温柔男主
- 禁止把整体年龄感画成熟
- 禁止把他画成标准古偶男主或仙侠美男
- 禁止把他画成纯反派、纯小丑或纯疯癫怪人
- 禁止把白衣画成飘飘欲仙的摆拍仙装
- 禁止脱离 `lh/cuidongshan/cds.png` 擅自换成另一套通用服装
- 禁止使用华丽金饰、高冠、大面积贵气纹样来替代人物辨识度

## 4. 最容易跑偏的三类情况

### 4.1 双人同场

- 双人图最容易把崔东山带偏成普通古风男主。
- 必须额外强调：另一角色只能是配角，不能污染崔东山身份。

### 4.2 夜景、逆光、压光

- 高压光线下最容易出现下颌变宽、年龄感变成熟、眼神变空。
- 必须额外强调：戏剧光线只能改气氛，不能重新设计五官。

### 4.3 轻仰视、轻俯视、动作镜头

- 角度一大，最容易把鼻口比例和下颌改坏。
- 必须额外强调：不要让透视、动作和镜头角度改变脸长比例、下颌宽度和年龄感。

## 7. 出图验收清单

每次新图生成后，都必须按以下顺序验收：

1. 第一眼是否还是 `lh/cuidongshan/cds.png` 那个人，而不是另一个古风男性。
2. 脸型是否仍然保持母版的窄长、收束、非圆非宽结构。
3. 眉、眼、鼻、口的位置关系是否与母版一致，尤其眼神是否仍然“聪明且难看透”，眉心红籽是否仍然小而凝、清楚可辨。
4. 年龄感是否仍然稳定在 `18-21` 岁。
5. 身形和服装是否仍然成立为 `lh/cuidongshan/cds.png` 的白衣少年读书人口径。
6. 气质是否仍然是外轻内重、清醒克制、孤独深情，而非普通男主化、反派化或小丑化。
7. 本次变化是否只发生在表情、动作、场景、景别和轻微光线关系。

只要第 1-5 项里有任一项不稳定，就直接判定废弃，不进入后续使用。

## 9. 补强素材优先级

### 9.1 当前缺口

崔东山目前已有：

- `cds.png`：通用全身母版
- `cds_16view_v1.png`：角度辅助母版

目前最缺的是：

- 脸部近景锁定母版
- 同镜头、同光线、同角度的表情差分页

这意味着当前“全身轮廓”和“角度方向”比“近距离脸部识别”更稳，所以一旦进入强表情或强动作，最容易漂向普通古风男主。

### 9.2 最高优先级

#### P1. `cds_face_close_v1`（待制作）

- 类型：单人脸部近景锁定母版
- 构图：肩部以上到胸口上缘的中近景
- 角度：正面略偏三分之二正
- 表情：平静、中性、不可爱化、不卖温柔
- 光线：中性柔光
- 背景：干净留白或极淡纸面感
- 用途：建立最硬的脸部识别锚点

#### P2. `cds_face_34_v1`（待制作）

- 类型：单人脸部三分之四视角母版
- 构图：肩部以上中近景
- 角度：左或右三分之四，建议先固定一个方向
- 表情：平静或若有所思
- 光线：中性柔光
- 用途：补足“半侧身 + 回望”最常用视角

#### P3. `cds_face_lowturn_v1`（待制作）

- 类型：单人脸部轻低头视角母版
- 构图：肩部以上中近景
- 角度：头轻低，视线轻抬
- 表情：若有所思 / 安静看人
