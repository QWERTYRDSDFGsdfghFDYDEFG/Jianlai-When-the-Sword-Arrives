# 陈平安固定脸部身份与当前服装标准

适用对象：`cpa/` 目录下陈平安的全部后续立绘、表情、动作版、章节版、场景版。

母版基准图：

- 脸部身份母版：`cpa_face_standard_01.png`
- 当前服装母版：`cpa_outfit_variant_06_warm_weather.png`

## 1. 目标定义

本标准的唯一优先目标，不是单独生成一张“更好看”的陈平安，而是建立陈平安的固定脸部身份标准，并在当前阶段锁定统一服装版本。

后续一切变化，必须建立在同一张脸、同一套当前服装的前提上进行：

- 脸型固定
- 五官比例固定
- 年龄感固定
- 气质底色固定
- 当前服装固定

允许变化的内容只有：

- 表情
- 动作
- 场景

禁止把每一张新图当成“重新设计一次陈平安的脸”或“重新设计一次陈平安的衣装”。

## 2. 五项硬锁定

以下五项是后续所有生成的最高优先级，任何一项漂移，都视为不合格。

### 2.1 脸型锁定

- 以 `cpa_face_standard_01.png` 为唯一脸型母版。
- 脸部整体为偏瘦长、收束明确的青年脸型。
- 额头、颧侧、下颌、下巴的收放关系必须保持一致。
- 下颌不能变宽，不能变圆，不能变成更尖锐的夸张 V 脸。
- 下巴长度和收束角度必须维持母版观感。

### 2.2 五官比例锁定

- 眉眼距离保持母版比例，不可随意拉近或拉远。
- 眼睛长度、开合度、眼尾走势、内外眼角关系保持母版比例。
- 鼻梁长度、鼻头位置、鼻尖落点保持母版比例。
- 人中长度、嘴宽、唇线厚薄和嘴角落点保持母版比例。
- 耳位、发际线与面部中轴关系不得随意改变。

原则：可以换表情，但不能换五官设计。

### 2.3 年龄感锁定

- 固定年龄感为 20-22 岁的年轻男子。
- 必须保留“青年已入世”的成熟度，不能画成少年感过强。
- 不能老化成 25 岁以上的沧桑脸，也不能幼化成 17-18 岁的稚气脸。
- 皮相应克制、自然，不要出现过度精修、过度艳化、过度偶像化处理。

### 2.4 气质底色锁定

陈平安的底色必须始终是：

- 沉静
- 可靠
- 自持
- 坚韧

允许在具体画面里出现：

- 轻微温和
- 疲惫
- 警觉
- 克制的怒意
- 压抑中的悲伤

但不允许整体气质改成：

- 轻佻
- 妖异
- 风流公子感
- 邪气反派感
- 过度柔媚
- 过度冷艳
- 过度热血少年漫主角感

原则：情绪可以波动，底色不能换人。

### 2.5 当前服装锁定

- 以 `cpa_outfit_variant_06_warm_weather.png` 为当前唯一服装母版。
- 当前阶段默认固定为该图的暖天行路装，不再允许自由更换其他服装版本。
- 必须保持该图的整体服装结构关系，包括：
  - 灰蓝色短袖外袍
  - 白色内衫
  - 深色长裤
  - 腰间多层绳带与小布袋
  - 小腿绑带
  - 草鞋
  - 背后行路架与随身行囊
- 允许出现轻微褶皱、污损、风尘感，但不允许改成另一套衣服、另一种剪裁、另一种配色方案。
- 不允许把当前服装替换成贵气长袍、飘逸仙侠装、厚重冬装、书生礼服或其他未批准版本。

原则：可以有使用痕迹，但不能换服装版本。

## 3. 允许变化与禁止变化

### 3.1 允许变化

- 表情变化：平静、凝神、迟疑、轻微温和、压抑、克制怒意、疲惫、警觉
- 动作变化：站立、侧身、回望、垂眸、抬手、执物、行走、静坐
- 场景变化：湖岸、雨雾、舟上、屋内、渡口、夜色、雪景、岸边泥地

### 3.2 禁止变化

- 禁止重新设计脸型
- 禁止改变五官间距和主比例
- 禁止把眼睛画大一号来换“精致感”
- 禁止把鼻子改短、改挺、改幼态来换“少年感”
- 禁止把嘴唇画得更厚、更艳、更有妆感
- 禁止把下颌改宽、改圆、改成陌生人
- 禁止把整体年龄感画年轻或画老
- 禁止把气质改成偶像风、美型风、妖俊风
- 禁止脱离 `cpa_outfit_variant_06_warm_weather.png` 另起一套服装
- 禁止擅自改外袍长度、袖型、内衫结构、裤型、鞋型、绑带体系
- 禁止把背后行路架改成书箱、背篓、木框背架以外的陌生结构
- 禁止把当前暖天行路装画成贵气、华丽、精修偶像风穿搭

## 4. 生成提示词总规则

后续任何陈平安出图，都必须默认附带以下约束：

```text
以 cpa_face_standard_01 为唯一脸部身份母版，以 cpa_outfit_variant_06_warm_weather 为唯一当前服装母版。
严格锁定陈平安的脸型、五官比例、年龄感、气质底色和当前服装版本。
禁止重新设计脸，禁止改变脸部骨相，禁止改变眉眼鼻口之间的相对比例。
禁止擅自更换服装版本，禁止偏离当前暖天行路装结构。
必须保持 20-22 岁青年感，气质沉静、可靠、自持、坚韧。
只允许修改表情、动作和场景，服装仍以 cpa_outfit_variant_06_warm_weather 为准。
如果新图更好看但不像 cpa_face_standard_01，或服装偏离 cpa_outfit_variant_06_warm_weather，也判定为失败。
```

## 5. 生成时的负面约束

后续生成时，应同时加入以下否定约束：

```text
不要幼态化，不要老化，不要娃娃脸，不要宽下颌，不要夸张 V 脸，不要大眼改脸，不要美妆感，不要偶像风，不要邪魅感，不要妖异感，不要重新设计五官，不要改变年龄段，不要更换服装版本，不要改成贵气长袍，不要改成仙侠飘带装，不要改成厚重冬装。
```

## 6. 出图验收清单

每次新图生成后，都必须按以下顺序验收：

1. 第一眼是否还是 `cpa_face_standard_01` 那个人，而不是“另一个长得像的古风男性”。
2. 脸型是否仍然保持母版的瘦长收束结构。
3. 眉、眼、鼻、口的位置关系是否与母版一致。
4. 年龄感是否仍然稳定在 20-22 岁。
5. 气质是否仍然是沉静、可靠、自持、坚韧，而非风格漂移。
6. 服装是否仍然是 `cpa_outfit_variant_06_warm_weather.png` 的当前固定版本。
7. 本次变化是否只发生在表情、动作、场景。

只要第 1-4 项里有任一项不稳定，或第 6 项服装跑偏，就直接判定废弃，不进入后续使用。

## 7. 文件与版本约定

- `cpa_face_standard_01.png` 是陈平安固定脸部身份母版。
- `cpa_outfit_variant_06_warm_weather.png` 是陈平安当前固定服装母版。
- 后续所有陈平安图，默认都应注明“基于该母版锁脸并锁服装生成”。
- 新图命名应表达“版本变化项”，而不是暗示“重新做人设”。

推荐命名思路：

- `cpa_standard_face_locked_exp_xx.png`
- `cpa_standard_face_locked_act_xx.png`
- `cpa_shujianhu_face_locked_xx.png`

## 8. 执行原则

结论只有一条：

陈平安后续所有图像工作，首先是在维护同一个人的身份连续性与当前服装连续性，其次才是做单张画面的表现力。
