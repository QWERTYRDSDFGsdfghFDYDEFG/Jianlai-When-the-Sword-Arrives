---

## 1. 使用顺序

1. 先提供 `lh/lbp/李宝瓶固定脸部身份与当前服装标准.md`
2. 再附上当前正式资产：
   - `lh/lbp/lbp_face_close_v1.png`
   - `lh/lbp/lbp_face_34_v1.png`
   - `lh/lbp/lbp_exp_sheet_v1.png`
3. 最后只补这次要变化的变量：表情 / 动作 / 场景 / 景别 / 光线

## 3. 最稳的起手值

- 表情：认真 / 微笑 / 疑惑

- 景别：半身 / 中近景
- 光线：中性柔光
