1. `lbp_face_close_v1`
2. `lbp_exp_sheet_v1`
3. `lbp_pose_small_v1`

## 3. 任务一：`lbp_face_close_v1`

### 3.1 任务目标

建立李宝瓶最硬的近景锁脸锚点。

这是本批次最优先的一张，也是后面表情页和动作页的基础。

### 3.2 输出定义

- 文件名：`lbp_face_close_v1.png`
- 类型：单人近景锁脸母版
- 推荐状态：先 `test` / `cand`，用户确认后再决定是否转正式

### 3.3 构图要求

- 肩部以上到胸口上缘
- 中近景
- 正面略偏三分之二正
- 中性柔光
- 简洁背景或极浅留白背景

### 3.4 表情要求

- 平静
- 认真
- 干净
- 不卖萌

### 3.5 重点锁定项

- 小脸
- 细下巴
- 面颊一点软感
- 清秀杏眼
- 齐刘海
- 高双髻
- 金铃发饰
- 红发带

### 3.6 提示词骨架

```text
这次不是重新设计角色，而是基于既有正式资产生成同一个李宝瓶的稳定近景锁脸图。当前正式参考是 lh/lbp/lbp_face_close_v1.png、lh/lbp/lbp_face_34_v1.png、lh/lbp/lbp_exp_sheet_v1.png。请严格保持书院读书时期的小姑娘李宝瓶身份：小脸、细下巴、下庭偏短但不圆萌，面颊有一点软感，清秀杏眼，黑发齐刘海，两侧高双髻，双髻旁有小巧金铃发饰，红发带自然垂下。她穿红色日常书院服，交领内层清楚，红色外层结构清楚，腰间系带克制整洁。
本次只做单人近景锁脸母版：肩部以上到胸口上缘的中近景，正面略偏三分之二正，中性柔光，简洁留白背景。表情平静、认真、干净，不卖萌。重点是把“小姑娘脸 + 齐刘海双髻 + 金铃红发带”稳定锁住。统一风格：东方幻想、宋代审美、半写实、沉静克制、低饱和、真实材质结构，1920x1080 横版。如果新图更好看但不像母版里的同一个李宝瓶，直接判定失败。
```

### 3.7 负面词骨架

```text
不要成熟少女，不要网红脸，不要偶像化古偶脸，不要超大圆眼，不要甜妹萌系，不要旅行装，不要江湖侠女，不要仙侠女主，不要复杂头饰，不要去掉齐刘海、高双髻、金铃发饰和红发带。
```

### 3.8 通过标准

- 第一眼就是同一个李宝瓶
- 近景下不漂成普通古风少女
- 小姑娘年龄感稳定
- 五官比例稳定
- 发型与头饰完整成立
- 近景里仍然克制、干净，不甜妹化

### 3.9 失败表现

- 漂成熟
- 漂萌系
- 眼睛过大
- 下巴结构消失
- 双髻或金铃不稳
- 红发带弱化或消失

### 3.10 失败回退

若失败，先回退调整下面变量，不要直接换大方案：

1. 收回表情强度
2. 回到更中性的光线
3. 收紧镜头变化
4. 加重“五官比例不得变化”的约束

### 3.11 完成后动作

- 先标为 `test` 或 `cand`
- 不自动回写标准文档
- 只有用户确认后，才考虑是否作为正式近景母版

### 3.12 是否允许进入下一张

只有当这张图通过标准后，才进入 `lbp_exp_sheet_v1`。

## 4. 任务二：`lbp_exp_sheet_v1`

### 4.1 任务目标

做第一张正式级表情差分页，锁定“同一个李宝瓶的脸如何变化”。

这张图的目标不是做最终立绘，而是稳定表情系统。

### 4.2 开始条件

必须先有通过验收的 `lbp_face_close_v1`。

### 4.3 输出定义

- 文件名：`lbp_exp_sheet_v1.png`
- 类型：近景正式表情差分页
- 推荐状态：先 `test` / `cand`，用户确认后再决定是否转正式

### 4.4 页面规格

- 建议 `6` 格或 `8` 格
- 全部使用同一个近景镜头
- 同角度
- 同光线
- 同背景
- 同发型
- 同服装露出范围

### 4.5 首批表情

1. 中性平静
2. 认真
3. 疑惑
4. 微笑
5. 小得意
6. 有点委屈

### 4.6 表情写法要求

每格表情都必须拆成：

- 眉
- 眼睑与视线
- 嘴角与嘴形
- 面颊与下颌张力
- 头部微角度

不能只写“认真”“疑惑”这种标签。

### 4.7 提示词骨架

```text
基于已经稳定通过的李宝瓶近景锁脸母版，制作正式表情差分页。要求同镜头、同角度、同光线、同背景、同发型和同服装露出范围，只改变表情结构，不改变脸型和五官比例。人物始终是同一个书院时期的小姑娘李宝瓶：小脸、细下巴、软感面颊、清秀杏眼、齐刘海、高双髻、金铃、红发带、红色日常书院服。
本页表情为：中性平静、认真、疑惑、微笑、小得意、有点委屈。每个表情必须落实到可见结构：眉、眼睑与视线、嘴角与嘴形、面颊与下颌张力、头部微角度。目标是“同一个李宝瓶的不同状态”，不是几张长得接近的小姑娘。统一风格：东方幻想、宋代审美、半写实、沉静克制、低饱和、真实材质结构，1920x1080 横版。
```

### 4.8 负面词骨架

```text
不要表情一变就换脸，不要眼睛大小忽大忽小，不要下巴长度变化，不要发型漂移，不要金铃和红发带忽有忽无，不要甜妹卖萌化，不要大笑夸张化，不要偶像海报感。
```

### 4.9 通过标准

- 整页第一眼就是同一个李宝瓶
- 六格之间五官比例稳定
- 表情差异明确，不糊成一类
- 小姑娘年龄感始终稳定
- 发型、金铃、红发带始终稳定

### 4.10 失败表现

- 每格像不同人
- 只有嘴在变，其他结构不变
- 一委屈就甜妹化
- 一微笑就偶像化
- 一疑惑就眼睛放大

### 4.11 失败回退

若失败，优先这样修：

1. 减少表情幅度
2. 缩回到 `6` 格
3. 固定头部微角度不要乱跳
4. 明确每格只变一个主表情方向

### 4.12 完成后动作

- 先标为 `test` 或 `cand`
- 不自动回写标准文档
- 只有用户确认“以后作为正式表情页”时才回写

### 4.13 是否允许进入下一张

只有当这张表情页能稳定证明“同一个李宝瓶的脸如何变化”后，才进入 `lbp_pose_small_v1`。

## 5. 任务三：`lbp_pose_small_v1`

### 5.1 任务目标

建立李宝瓶最常用的轻动作系统，让她在 Ren'Py 里不再只能用全身标准站姿。

这张图只解决小动作，不解决大动作。

### 5.2 开始条件

必须先有通过验收的：

1. `lbp_face_close_v1`
2. `lbp_exp_sheet_v1`

### 5.3 输出定义

- 文件名：`lbp_pose_small_v1.png`
- 类型：轻动作页
- 推荐状态：先 `test` / `cand`，用户确认后再决定是否转正式

### 5.4 页面规格

- 建议 `4` 格
- 同服装
- 同镜头逻辑
- 优先中景或三分之四身
- 只改重心、头部、手势与轻微跟随关系

### 5.5 首批动作

1. 轻回头
2. 轻提手
3. 认真听
4. 倔强站定

### 5.6 动作写法要求

每格至少写清：

- 身体朝向
- 重心在哪条腿
- 双手在做什么
- 头部朝向
- 肩线 / 骨盆是否有轻微偏转
- 双髻、发带、衣摆是否有轻微跟随

### 5.7 提示词骨架

```text
基于已经稳定通过的李宝瓶锁脸母版和表情系统，制作李宝瓶轻动作页。人物始终是同一个书院时期的小姑娘李宝瓶：小脸、细下巴、清秀杏眼、齐刘海、高双髻、金铃、红发带、红色日常书院服。只做小动作，不做大动作，不武侠化，不卖萌摆拍，不成熟化。统一镜头逻辑下做四个动作：轻回头、轻提手、认真听、倔强站定。
每个动作都必须有明确重心和方向：说明身体朝向、重心落在哪条腿、双手在做什么、头部转向哪里、肩线和骨盆是否轻微偏转、双髻红发带和衣摆有没有自然跟随。整体仍然是书院时期的小姑娘日常状态，适合 Ren'Py 中景和半身复用。统一风格：东方幻想、宋代审美、半写实、沉静克制、低饱和、真实材质结构，1920x1080 横版。
```

### 5.8 负面词骨架

```text
不要大跳，不要大跨步冲刺，不要舞台式挥手，不要偶像卖萌手势，不要大幅扭腰，不要武侠起势，不要双手同时做复杂动作，不要把小动作做成大表演。
```

### 5.9 通过标准

- 四格都是同一个李宝瓶
- 动作比标准站姿更活，但不夸张
- 每个动作有明确重心和方向
- 听人、回头、提手、倔强站定四种状态清楚可分
- 适合后续 Ren'Py 中景复用

### 5.10 失败表现

- 动作一大就换脸
- 手势复杂但重心不成立
- 一认真听就僵成站桩
- 一轻提手就卖萌
- 一倔强站定就成熟化

### 5.11 失败回退

若失败，优先这样修：

1. 把动作幅度减半
2. 减少手势复杂度
3. 回到更稳的中景
4. 重新强调“先有重心，再有动作”

### 5.12 完成后动作

- 先标为 `test` 或 `cand`
- 不自动回写标准文档
- 只有用户确认“以后作为正式动作页”时才回写

## 6. 本批次的停止条件

满足以下情况之一，就先停止往后扩：

1. 三张里任一张连续多轮都稳不住身份
2. 表情页仍然一变就换脸
3. 动作页仍然一动就成熟化

出现以上情况时，不进入更多场景图，不先补大动作，不先补群像。
