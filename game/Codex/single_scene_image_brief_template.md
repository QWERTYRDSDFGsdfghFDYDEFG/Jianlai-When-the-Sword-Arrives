# 单场出图填写模板

适用范围：本项目任意单场戏的角色图、动作图、事件图、背景图、CG 图需求整理。

用途：把“这一场到底要画什么、为什么要画、画到什么程度算对”一次写清，减少角色漂移、构图跑偏、出图后无法接入 Ren'Py。

使用方式：

1. 先读 `Codex/asset_pipeline_sop.md`
2. 再按本模板逐项填写
3. 填完后再进入提示词编写或出图阶段
